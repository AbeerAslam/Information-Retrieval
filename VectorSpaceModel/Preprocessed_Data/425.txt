student retention pattern prediction employing linguistic feature extracted from admission application essay educational data mining student retention application essay linguistic feature predictive analysis this paper investigates use of linguistic feature extracted from application essay of student enrolled university academic program their retention pattern prediction three set of linguistic feature generated from text analysis 1 latent dirichlet allocation lda based topic modeling with variety of topic number 2 linguistic inquiry and word count liwc and 3 part of speech po distribution various classification experiment implemented evaluate prediction performance of student retention pattern from these three feature set and their combination result show that po distribution feature yield best prediction performance among these three while neither lda feature nor ensemble method improves predictive performance which is contrary admission expert manual analysis method conventional admission process
